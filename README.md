# Quiz App

## Description
The Quiz App is a Python-based application that quizzes users on various topics. It reads questions and answers from a predefined set of data, presents them to the user, and keeps track of their score. The application is built using the Tkinter library for the graphical user interface and SQLite for data storage.

## Features
- User authentication: Users are prompted to enter their name before starting the quiz.
- Dynamic question presentation: Questions are dynamically loaded from the data set and presented to the user.
- Multiple-choice selection: Users can select their answer from a set of multiple-choice options.
- Real-time feedback: Users receive immediate feedback on whether their answer is correct or incorrect.
- Scoring system: The application keeps track of the user's score throughout the quiz.
- Database integration: User answers are stored in an SQLite database for future reference.

## Installation
1. Clone the repository: `git clone https://github.com/your-username/quiz-app.git`
2. Navigate to the project directory: `cd quiz-app`
3. Install dependencies: `pip install ttkbootstrap`

## Usage
1. Run the application: `python quiz_app.py`
2. Enter your name in the provided field and click "Submit" to start the quiz.
3. Answer the questions presented and receive immediate feedback.
4. Click "Next" to proceed to the next question.
5. Once the quiz is completed, a message will display the final score.
